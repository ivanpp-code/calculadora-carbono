💻 Sobre o Projeto

Calculadora EcoTrip é um projeto que simula o impacto ambiental de viagens, estimando a emissão de carbono com base em fatores como distância, meio de transporte e perfil do trajeto, ajudando o usuário a tomar decisões mais conscientes e sustentáveis.

📚 Resumo das minhas atividades

A Calculadora é funcional, uma página web feita em js e rodando 100%; é uma versão simplificada da desenvolvida pelo Pablo no curso. Ela não possui a parte de créditos de carbono, e troquei o "caminhão" por "motocicleta".

Como eu a desenvolvi:
1) a estrutura do pacote, assim como os arquivos index.html, style.css e routes-data.js eu fiz via prompt do GitHub Copilot, dentro do Visual Studio Code; eu usei prompts praticamente iguais ao do Pablo, fui copiando e adpatando, pelo que via nas telas, pelo curso
2) os arquivos calculator.js, config.js e ui.js, eu copiei boa parte do git do Pablo. A partir disso, fiz tudo "sozinho" junto com o Copilot
3) fui perguntando no prompt como "conectar" os arquivos, encontrava e corrigia erros (o próprio Copilot identificou a necessidade de preencher o arquivo app.js, que estava vazio e o preencheu), e com o tempo fui refinando o backend e o frontend, até tudo ficar funcional
